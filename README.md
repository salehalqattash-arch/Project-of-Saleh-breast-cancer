## Overview | نظرة عامة

نظام ذكاء اصطناعي متكامل للكشف عن الأورام باستخدام تقنيات **Machine Learning** و **Deep Learning**.
يدعم النظام طريقتين للتشخيص:

1. **التشخيص الرقمي**: باستخدام 7 خصائص سريرية
2. **التشخيص بالصور**: باستخدام صور الخزعة (Biopsy Images)

> An AI-powered tumor detection system using ensemble ML models and CNN for image analysis.

## Model Performance | أداء النماذج

| ------------------- | -------- | -------- |

| Gradient Boosting   | 95.0%    | 0.971    |
| SVM                 | 94.0%    | 0.965    |
| Logistic Regression | 92.5%    | 0.950    |
| **CNN (Image)**     | **85%**  | **0.89** |
